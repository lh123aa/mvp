# Demand Miner - 需求挖掘与最小商业闭环分析

> 🏭 **需求挖掘与最小商业闭环全适配系统**  
> 帮助用户从陌生领域探索到最小商业闭环落地

## 项目简介

Demand Miner 是一个基于 iFlow CLI 的智能需求挖掘技能（Skill），通过结构化的分析方法论，帮助用户：
- 深入分析用户痛点和未被满足的需求
- 设计低成本、可快速验证的商业闭环
- 精准定位目标用户群体
- 根据约束条件推荐最优解决方案

## 快速开始

### 触发方式

在 iFlow CLI 中输入以下关键词即可触发：
```
产品分析、项目分析、商业分析、商业闭环、需求挖掘、商业模式、创业分析、副业规划
```

### 使用流程

1. **触发技能**：输入关键词
2. **前置对齐**：回答4个问题构建用户画像
3. **选择模式**：根据认知层级进入对应分析模式
4. **获取方案**：获得定制化的需求分析与商业闭环方案

## 功能特性

### 🎯 前置对齐系统
通过4个维度构建精准用户画像：
- 核心目标
- 核心目的（副业/创业/学习/转型）
- 认知层级（陌生/基础/资深）
- 预算周期约束

### 🔄 三种分析模式

| 模式 | 适用人群 | 服务方式 |
|------|----------|----------|
| 落地顾问 | 完全陌生 | AI主导，直接输出完整方案 |
| 引导教练 | 有基础认知 | 引导式提问，共同完成分析 |
| 深度共创 | 资深从业者 | 深度探讨，挖掘个性化需求 |

### ⚡ 约束过滤
- 预算约束：0成本 / 5000元内 / 2万元内
- 周期约束：1个月 / 3个月 / 6个月

### 📊 行业数据
覆盖多个热门行业的用户画像、痛点、方案和验证指标：
- 跨境电商
- AI服务
- 短视频带货
- 在线教育
- 私域电商
- 本地生活
