## 对话复盘记录

每次对话结束后自动记录，用于系统迭代优化。

### 复盘记录格式

```markdown
## [日期] - 对话复盘

### 对话概况
- 核心目标：[目标名称]
- 认知层级：[层级]
- 选用模式：[模式]
- 对话轮次：[N轮]

### 系统响应评估

| 评估项 | 评分 | 说明 |
|--------|------|------|
| 前置对齐 | ✓/✗ | 描述 |
| 模式执行 | ✓/✗ | 描述 |
| 内容质量 | ✓/✗ | 描述 |
| 约束遵守 | ✓/✗ | 描述 |

### 发现问题
1. [问题描述]

### 迭代建议
1. [建议描述]

### 下次改进重点
[改进项]
```

> 注：复盘记录将随版本更新持续补充
